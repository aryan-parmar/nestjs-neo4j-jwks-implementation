## Note

- replace private and public key in key folder and replace jwks in ./public/.well-known/jwks.json
- test api service in ./test-api-service

## Description

JWKS implementation in nestjs using neo4j as database

## Installation

```bash
#yarn
$ yarn install

#npm 
$ npm install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
